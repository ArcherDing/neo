First what you have to do is to install and setup Go.
You can find detail explanation how to do that on [https://golang.org/doc/install](https://golang.org/doc/install).

After you have setup your Go installation, execute following command to get Neo installation.

```Bash
go get github.com/ivpusic/neo
```

Neo commes with nice CLI tool. To install it run following command:

```Bash
go get github.com/ivpusic/neo/cmd/neo
```

That's it. You are ready to use Neo. Look at <a href="{{site.url}}/tutorials/2015/01/22/introduction.html">Introduction example</a>.
