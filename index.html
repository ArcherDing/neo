Neo is minimal and fast Go Web Framework with extremely simple API.

During development you will enjoy in automatic reruning and recompiling your Neo application when source changes.

Build your Neo Application in few lines of code.

{% highlight go %} package main import ( "github.com/ivpusic/neo" ) func main() { app := neo.App() app.Get("/", func(ctx *neo.Ctx) { ctx.Res.Text("I am Neo Programmer", 200) }) app.Start() } {% endhighlight %} Run it! {% highlight bash %} neo run main.go {% endhighlight %} That's it. You did it. Navigate to localhost:3000 to see result.
